# Bank Account
A Python class implementing core banking operations with transaction history and input validation.

## Setup
1. Clone the repository
2. Ensure Python 3 is installed
3. No external dependencies required

## Run
```
python bank_account.py
```

## What I learned
How to use encapsulation and instance methods in Python OOP, implement input validation with `ValueError`, and maintain ordered transaction history across multiple operations.
